# Guía de Colores Premium - SIGA

Esta guía detalla la paleta de colores corporativa y "premium" utilizada en el sistema para su aplicación en entornos de producción.

## 1. Paleta Premium Principal (Verdes)

Estos colores definen la estética "Arizar Premium" del sistema.

| Variable | Hex Code | Uso Sugerido |
| :--- | :--- | :--- |
| **Premium Dark** | `#022c22` | Fondos muy oscuros, sidebars en modo oscuro, texto fuerte. |
| **Premium Green** | `#059669` | Color de marca principal, botones primarios, iconos activos. |
| **Premium Light** | `#34d399` | Acentos claros, hovers, badges de éxito suaves. |
| **Premium Gold** | `#d4af37` | Detalles de lujo, medallas, botones especiales de "recompensas" o admin. |

## 2. Paleta Funcional (Shadcn/UI & Tailwind)

Utilizada para la estructura y elementos interactivos base.

| Propiedad | Hex Code | Descripción |
| :--- | :--- | :--- |
| **Primary** | `#009900` | El verde puro utilizado en el "Ring" y elementos de enfoque (focus). |
| **Background** | `#f8fafc` | Fondo general de la aplicación (Slate 50). |
| **Foreground** | `#0f172a` | Color de texto principal (Slate 900). |
| **Ring** | `#00cc00` | Color de brillo/anillo en inputs activos. |

## 3. Escala de Esmeralda (Emerald)

Utilizada para degradados (Gradients) y estados de carga.

- **Emerald 500**: `#10b981` (Verde estándar)
- **Emerald 600**: `#059669` (Esmeralda corporativo - **El más usado**)
- **Emerald 900**: `#022c22` (Verde profundo para sombras y profundidad)

---

## 4. ¿Qué "pintar" con qué?

- **Botones de Acción**: Usa `#059669` (Premium Green).
- **Cabeceras (Headers)**: Usa un degradado de `#009900` a `#059669`.
- **Textos de Títulos**: Usa `#022c22` (Premium Dark) para máxima legibilidad y sobriedad.
- **Bordes y Divisores**: Usa `#e2e8f0` para mantener la interfaz limpia.
